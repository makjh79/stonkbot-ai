# Social Media Credibility Guide for Stock Sentiment

## Tier 1 — Most Reliable 🥇

### Twitter/X (Fintwit)
| Aspect | Details |
|--------|---------|
| **Strength** | Real analysts, fund managers, institutional traders post real-time thoughts |
| **Best For** | Breaking news, earnings reactions, macro commentary, smart money positioning |
| **Who to Follow** | Verified accounts, CFA/CFP holders, fund managers with track records |
| **⚠️ Caveat** | Filter aggressively — ignore anonymous pumpers and "DM for alpha" accounts |
| **Example** | @fundstrat, @charliebilello, @biancoresearch |

### StockTwits
| Aspect | Details |
|--------|---------|
| **Strength** | Dedicated trader community, built-in sentiment scoring, real-time ticker streams |
| **Best For** | Crowd positioning, contrarian signals, options flow discussion |
| **Key Metric** | Bullish/Bearish ratio per ticker |
| **⚠️ Caveat** | Heavy retail bias, echo chamber risk — when everyone agrees, be skeptical |
| **Example** | $HOOD stream, sentiment indicator |

---

## Tier 2 — Useful but Noisy 🥈

### Reddit (WallStreetBets, r/stocks, r/investing)
| Aspect | Details |
|--------|---------|
| **Strength** | Early trend detection, options flow chatter, retail positioning |
| **Best For** | Detecting FOMO before it hits mainstream, gamma squeeze potential |
| **Key Signal** | Unusual mention volume spikes, YOLO posts with real position screenshots |
| **⚠️ Caveat** | Mostly retail, YOLO culture, usually late to smart money moves |
| **When to Trust** | When Reddit *hates* something = often contrarian buy signal |

### YouTube (Stock Commentary)
| Aspect | Details |
|--------|---------|
| **Strength** | Technical analysis deep-dives, earnings model walkthroughs, educational content |
| **Best For** | Learning thesis, understanding bear/bull cases, chart analysis |
| **Who to Trust** | Channels with track records, transparent P&L, no clickbait thumbnails |
| **⚠️ Caveat** | Ad-driven clickbait, varying quality, lagging indicator (takes time to produce) |
| **Red Flags** | "This stock will 10x by Friday," constant alarmist titles |

### Discord (Trading Communities)
| Aspect | Details |
|--------|---------|
| **Strength** | Real-time alerts, algo signals, tight-knit trader groups |
| **Best For** | Day trading signals, earnings plays, options flow bots |
| **⚠️ Caveat** | Scam servers everywhere, pump & dump groups, paid "alpha" rarely worth it |
| **Rule** | Never pay for Discord access — real edge isn't sold for $29/month |

---

## Tier 3 — Entertainment / Lagging 🥉

### TikTok / FinTok
| Aspect | Details |
|--------|---------|
| **Strength** | Viral sentiment indicator, pure retail FOMO gauge |
| **Best For** | Knowing when to *exit* — if your taxi driver is talking about it... |
| **⚠️ Caveat** | Usually 24-48 hours late, exit liquidity warnings, meme-driven |
| **Reality** | By the time it hits FinTok, smart money is already positioned |
| **Use Case** | Counter-indicator — TikTok hype = time to take profits |

---

## Summary: How to Use This

```
TRUST  →  Twitter verified analysts + StockTwits sentiment
CONFIRM →  Reddit mention spikes + YouTube thesis alignment  
FADE   →  TikTok hype (usually the top)
RESPECT →  Reddit hate (often contrarian signal)
```

## Red Flags Across All Platforms

🚩 "Guaranteed 10x"  
🚩 "DM me for the real play"  
🚩 No position proof (screenshots or it didn't happen)  
🚩 Urgency — "Buy before market open or miss out"  
🚩 Emoji overload in serious analysis  
🚩 Anonymous account with no track record

---

*Remember: Social sentiment tells you* ***what the crowd is doing*** — *not what you should do. Use it as context, not a strategy.*
