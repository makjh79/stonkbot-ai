### Key lessons this session
- SSH heredoc quoting is unreliable for Python scripts — write locally, scp, then run
- Agent sessions time out on long SSH waits (>5 min) — use background processes and check back
- Rotation logic must check tradability BEFORE trimming, not after — otherwise it sells into a failed buy
- Systemd Restart=always saved us when the bot crashed after the PSTG 422 error
- `su -s /bin/bash stonkai -c` spawns a process but doesn't set up the full environment — prefer systemd service for daemons
- When multiple processes write the same file as different users, file ownership keeps flipping — fix at the systemd service level, not just chown
- Always convert UTC server timestamps to HKT when reporting to Howie — he's in Hong Kong
- Trade log sync and bot rationale logging must use the same format, or the sync overwrites the bot's reasoning
