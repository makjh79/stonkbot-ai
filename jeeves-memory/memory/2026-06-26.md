### Lessons Learned
- Always chown JSON files to stonkai after manual root writes — bot can't write to root-owned files
- SSH heredoc quoting is unreliable for Python scripts — write locally, scp, then run
- `fetch_ai_watchlist.py` runs independently and can clobber brain-managed watchlist if not patched
- Yahoo Finance 429 rate-limits kick in fast when fetching 75 symbols sequentially — need batch or synthetic fallback
- Alpaca data API returns 401 on IEX feed for some symbols on free tier — paper keys have limited data access
- Sub-agent output limits: large builds can exceed sub-agent output budget — handle directly when they fail
- JS patching via Python string replacement is fragile — always verify the exact string exists before replacing
